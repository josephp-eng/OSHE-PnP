Notes
  • Designed for KiCad “Footprint positions” files (e.g., headers like “## Unit = mm, Angle = deg.”).
  • Columns expected: Ref Val Package PosX PosY Rot Side. Comment lines (#) are ignored.
  • Rotation is ignored (single orientation toolhead). You can enable it later when you add an orientation axis.
  • Flip Y defaults on because many PCB exports have negative Y; Ender bed coords are typically positive Y up.
